# AI Impact on Students - EDA

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on a dataset of **50,000 students** to understand the impact of Generative AI tools on academic performance, study habits, skill retention, anxiety, and burnout.

## 🎯 Objective

- Analyze AI usage among students.
- Identify factors affecting GPA and burnout.
- Generate insights to support responsible AI adoption in education.

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📊 Dataset

- **Rows:** 50,000
- **Columns:** 16
- **Missing Values:** None
- **Duplicate Values:** None

## 📈 Key Insights

- Moderate AI usage is associated with better academic performance.
- Excessive AI dependency may increase burnout risk.
- Traditional study habits remain important.
- Prompt engineering skills positively influence learning outcomes.
