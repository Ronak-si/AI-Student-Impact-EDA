# 🎓 AI Impact on Students - Exploratory Data Analysis

An Exploratory Data Analysis (EDA) project that investigates how Generative AI tools influence students' academic performance, study habits, skill retention, anxiety, and burnout using a dataset of **50,000 student records**.

## 📌 Project Overview

The rapid adoption of AI tools such as ChatGPT, Gemini, and GitHub Copilot has transformed education. This project analyzes student behavior and academic outcomes to understand the benefits and challenges of AI-assisted learning through meaningful visualizations and data-driven insights.

## 🎯 Objectives

- Analyze AI usage among students.
- Study the relationship between AI usage and academic performance.
- Examine the impact of AI on burnout, anxiety, and skill retention.
- Generate insights to support responsible AI adoption in education.

## 📊 Dataset Information

- **Records:** 50,000
- **Features:** 16
- **Missing Values:** None
- **Duplicate Records:** None

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📈 Exploratory Data Analysis

The project includes:

- Data Cleaning & Validation
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis
- Correlation Heatmap
- Pair Plot
- Business Insights & Recommendations

More than **15 meaningful visualizations** were created to understand relationships between AI usage, GPA, study habits, burnout, and student well-being.

## 🔑 Key Findings

- Moderate AI usage is associated with better academic performance.
- Excessive AI dependency may increase burnout risk.
- Traditional study habits continue to play an important role.
- Prompt engineering skills contribute to more effective AI usage.
- Institutional AI policies influence student learning behavior.
